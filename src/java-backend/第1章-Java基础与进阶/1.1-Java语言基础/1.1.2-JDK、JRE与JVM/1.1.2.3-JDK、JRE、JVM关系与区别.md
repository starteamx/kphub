# JDK、JRE、JVM关系与区别

## 三者的定义与职责

- JDK：Java开发工具包，面向开发者的完整工具集
- JRE：Java运行环境，面向用户的运行时环境
- JVM：Java虚拟机，执行字节码的核心引擎
- 包含关系：JDK包含JRE，JRE包含JVM

## 组件构成对比

- JDK组件：编译器、调试器、文档工具、JRE等
- JRE组件：类库、JVM、支持文件等
- JVM组件：类加载器、执行引擎、运行时数据区等
