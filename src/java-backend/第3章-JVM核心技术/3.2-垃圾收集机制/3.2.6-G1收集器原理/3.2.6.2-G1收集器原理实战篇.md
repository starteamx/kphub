# G1收集器原理实战篇

## G1收集器的启用与配置
- -XX:+UseG1GC：启用G1收集器
- 堆内存大小设置(-Xms, -Xmx)
- Region大小设置(-XX:G1HeapRegionSize)
- 停顿时间目标(-XX:MaxGCPauseMillis)
- 并发线程数控制(-XX:ConcGCThreads)
