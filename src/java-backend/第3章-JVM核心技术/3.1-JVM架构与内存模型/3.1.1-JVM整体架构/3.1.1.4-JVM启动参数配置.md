# JVM启动参数配置

## 参数分类与语法

- 标准参数(-开头)
- 非标准参数(-X开头)
- 高级参数(-XX开头)
- 参数语法规则与格式
- 参数优先级与覆盖关系

## 内存相关参数

- 堆内存配置(-Xms/-Xmx)
- 新生代与老年代比例(-XX:NewRatio)
- 永久代/元空间配置(-XX:MaxMetaspaceSize)
- 线程栈大小设置(-Xss)
- 直接内存配置(-XX:MaxDirectMemorySize)

## GC相关参数

- 垃圾收集器选择参数
- GC日志配置参数
- GC触发阈值调整
- 并行GC线程数设置
- GC性能调优参数

- 即时编译器选择
- 编译阈值设置
- 编译线程数配置
- 方法内联控制
- 逃逸分析与锁优化
