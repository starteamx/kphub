# 双亲委派模型详解

- 委派加载的基本流程
- loadClass方法的实现分析

## 双亲委派的优势

- 避免类的重复加载
- 保证Java核心API的安全性
- 保证类的唯一性
- 实现沙箱安全机制
- 类的命名空间隔离
