# 运行时数据区监控

## JDK自带监控工具

- jstat命令详解与使用
- jmap内存映射工具
- jstack线程栈分析
- jcmd多功能命令工具
- jinfo查看JVM参数

## 可视化监控工具

- JVisualVM使用技巧
- Java Mission Control(JMC)
- VisualGC插件应用
- JProfiler内存监控
- Eclipse Memory Analyzer(MAT)

## 监控指标与分析

- 堆内存使用率监控
- GC频率与时间分析
- 线程数量与栈使用情况
- 类加载与卸载监控
- 代码缓存使用情况

- JMX监控接口应用
