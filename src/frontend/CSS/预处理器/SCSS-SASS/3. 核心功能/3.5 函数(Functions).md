# SCSS/SASS函数(Functions)

Sass提供了丰富的内置函数，也支持自定义函数：

```scss
// 自定义函数
@function calculate-width($col-span, $total-cols: 12) {
  @return percentage($col-span / $total-cols);
}

.sidebar {
  width: calculate-width(3); // 25%
}

.main-content {
  width: calculate-width(9); // 75%
}
```
