# 源码调试与Source Maps

## 浏览器中调试Sass源码

Source Maps允许浏览器开发工具将编译后的CSS映射回原始的Sass源文件，便于调试：

```bash
# 生成带有Source Maps的CSS
sass --source-map input.scss output.css
```

## 配置Source Maps

在项目中配置Source Maps：

```scss
// 在Sass文件中控制Source Maps
@debug "This message will appear in the console";
```
