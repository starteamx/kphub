# 性能优化策略

## 避免嵌套过深

过深的选择器嵌套会导致CSS选择器过长，影响性能：

```scss
// 不推荐
.container {
  .header {
    .navigation {
      .menu {
        .item {
          a {
            color: red;
          }
        }
      }
    }
  }
}

// 推荐
.container .menu-item-link {
  color: red;
}
```

## 减少@extend滥用

过度使用@extend可能导致生成大量的选择器组合：

```scss
// 谨慎使用@extend
%button {
  padding: 10px 15px;
  border-radius: 4px;
}

.primary-button {
  @extend %button;
  background-color: #007bff;
}
```
