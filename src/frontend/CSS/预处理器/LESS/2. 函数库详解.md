# LESS函数库详解

## 字符串函数

LESS提供了多种处理字符串的函数：

```less
@escaped: escape("a=1");
// 结果: a%3D1

@unescaped: e("a=1");
// 结果: a=1

@replaced: replace("Hello, Mars", "Mars", "World");
// 结果: Hello, World
```

## 列表函数

LESS提供了处理列表的函数：

```less
@list: "a", "b", "c";
@length: length(@list); // 结果: 3
@item: extract(@list, 2); // 结果: "b"
```
