# LESS基础语法与核心特性

## 快速入门

### LESS简介

LESS(Leaner Style Sheets)是一种CSS预处理语言，它扩展了CSS，增加了变量、混合、函数等特性。LESS可以运行在客户端或服务器端。

## 核心语法

### 变量系统

LESS使用@符号定义变量：

```less
@primary-color: #007bff;

.button {
  background-color: @primary-color;
  color: white;
}
```
