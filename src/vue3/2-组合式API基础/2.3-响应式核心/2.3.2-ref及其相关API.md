# ref及其相关API

## ref基础
- 创建响应式引用
- 访问和修改值
- 在模板中的自动解包
- 在响应式对象中的解包规则

## 相关API
- isRef：检查值是否为ref对象
- unref：获取ref值或非ref值
- toRef：为响应式对象的属性创建ref
- toRefs：将响应式对象转换为普通对象，其中每个属性都是指向原始对象相应属性的ref
- customRef：创建自定义ref，显式控制依赖追踪和更新触发
- shallowRef：创建一个浅层响应式ref

## 实际应用场景
- 表单处理
- 状态管理
- 异步数据获取
